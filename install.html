Installation Instructions

How to Install and Use the Demo

  1. Download the Demo.zipDemo.zip file (stored in the same directory).
  2. Unpack the ZIP file.
  3. Right-click on jinx64.xll and select Properties.
  4. In the Properties window, check the box labeled Unblock (if present), then click OK.
  5. Open example.xlsx in Microsoft Excel.
  6. Go to File → Options → Add-ins.
  7. At the bottom of the Add-ins window, select Excel Add-ins and click Go....
  8. Click Browse and navigate to the jinx64.xll file.
  9. Select it and confirm to load the add-in.
